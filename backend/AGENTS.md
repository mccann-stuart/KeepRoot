## Commands

| Command | Purpose |
|---------|---------|
| `npx wrangler dev` | Local development |
| `npx wrangler deploy` | Deploy to Cloudflare |
| `npx wrangler types` | Generate TypeScript types |

Run `wrangler types` after changing bindings in wrangler.jsonc.

## Security Review Checklist

- Reuse the shared safe-URL validator for every Worker-side fetch path, including redirects, source feeds, queued jobs, and image hydration.
- Keep authenticated API, auth, and MCP responses uncacheable.
- The dashboard service worker may cache static assets only; do not cache bearer-authenticated API responses.
- Scope every D1 read or write by `user_id` unless the table is explicitly global metadata.
- Return validation errors as 400 responses and avoid leaking internal exception details to clients.
